# Expense Tracker Application

## Overview
Expense Tracker is a web application that helps users manage their income and expenses. Users can add, view, and delete transactions while tracking their current balance.

## Features
- Add Income
- Add Expenses
- Delete Transactions
- Real-time Balance Calculation
- Data Stored in Local Storage

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

## How to Run
1. Download the project.
2. Open index.html in your browser.
3. Start adding transactions.
